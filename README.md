# Data Analytics - Spotify Scraping Project

## Overview
This project scrapes Hasan Raheem's tour data and analyzes it using SQL, Python, and Excel as part of my Spotify portfolio case study.

## Files
- `scraper.py` → Python script for scraping tour data
- `hasan_raheem_tour.csv` → Raw dataset of tour dates and venues
- `queries.sql` → SQL queries for practice and analysis
- `dashboard.xlsx` → Excel dashboard with charts and insights

## How to Use
1. Run `scraper.py` to generate the CSV file.
2. Import `hasan_raheem_tour.csv` into MySQL or Excel.
3. Use `queries.sql` to explore the dataset.
4. Open `dashboard.xlsx` to view visualizations.

## Skills Demonstrated
- Web scraping with Python (BeautifulSoup, Pandas)
- SQL queries for filtering, grouping, and joining
- Excel dashboards for visualization
- GitHub portfolio development

## Purpose
This case study benchmarks Hasan Raheem’s tour data for diaspora and mainstream audience expansion.

## Artistic Connection
Hasan Raheem’s music blends indie/lo-fi vibes with mainstream appeal, and his career shows how structured marketing campaigns can expand reach across diaspora and global audiences.  

My own work as **Sarah Afreen Malik** — DJ, vocalist, and producer — shares a similar bilingual (Urdu/English) style and indie sensibility. While Hasan often leverages marketing strategies, I focus on **organic growth** — building my fanbase through grassroots reach, authentic engagement, and community-driven discovery.  

This project highlights not only the technical side of scraping and analyzing his tour data, but also the artistic parallels between his journey and mine. It reflects how data analytics can support both mainstream marketing strategies and organic audience-building approaches.

## Tour Insights
Hasan Raheem’s North America and Europe tour revealed both strengths and gaps:

- ✅ Major cities like San Francisco, New York, Toronto, and London were covered.
- ❌ Diaspora-heavy hubs such as Chicago suburbs, Vancouver, Birmingham, and Manchester were missed.
- ⚠️ Not every venue sold out (Dallas, Houston, Cambridge), showing uneven demand.
- 🔁 San Francisco was repeated, suggesting strong Bay Area demand but limited promoter flexibility.
- ⭐ As one of Pakistan’s biggest pop stars, every show should have been sold out — the fact that some weren’t highlights missed opportunities in marketing and promotion.

These insights show how structured marketing secured big venues but overlooked diaspora-heavy regions and failed to maximize demand. In contrast, my own organic strategy focuses on reaching fans in overlooked cities and building authentic connections that drive consistent engagement.
