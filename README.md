# Data Analytics - Spotify & Instagram Case Study

## Overview
This project scrapes Hasan Raheem's tour data and combines it with Instagram analytics (@Sarahafreenmalik) to demonstrate how social engagement and streaming/touring insights can be connected. While Hasan’s tour serves as the dataset, the framework is designed to be applied to **local and emerging artists** — and eventually to my own bilingual (Urdu/English) music.

## Files
- `scraper.py` → Python script for scraping tour data
- `hasan_raheem_tour.csv` → Raw dataset of tour dates and venues
- `queries.sql` → SQL queries for practice and analysis
- `dashboard.xlsx` → Excel dashboard with charts and insights
- `instagram_insights.csv` → Exported Instagram analytics (followers, views, interactions)

## Skills Demonstrated
- Web scraping with Python (BeautifulSoup, Pandas)
- SQL queries for filtering, grouping, and joining
- Excel dashboards for visualization
- Social media analytics integration
- GitHub portfolio development

## Purpose
The goal is to show how data analytics can benchmark artist performance, identify missed opportunities, and connect **social traction (Instagram)** with **streaming/touring demand (Spotify)**. By analyzing one artist’s tour and my own Instagram insights, I demonstrate a framework that can be scaled to help **other artists** — especially local voices — connect with audiences worldwide.

## Tour Insights (Case Study)
Hasan Raheem’s North America and Europe tour revealed both strengths and opportunities:

- ✅ Major cities like San Francisco, New York, Toronto, and London were covered.
- ❌ Diaspora-heavy hubs such as Chicago suburbs, Vancouver, Birmingham, and Manchester were missed — places where demand suggests he could have sold out shows.
- ⚠️ Not every venue sold out (Dallas, Houston, Cambridge), showing uneven demand across regions.
- 🔁 San Francisco was repeated, suggesting strong Bay Area demand but limited expansion to nearby cities like San Jose or Sacramento.
- ⭐ As one of Pakistan’s biggest pop stars, Hasan Raheem had the potential to sell out every show. Highlighting missed cities and unsold venues points to opportunities for future optimization.
