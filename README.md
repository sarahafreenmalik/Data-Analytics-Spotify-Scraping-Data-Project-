# Data Analytics - Spotify Scraping Project

This project scrapes Hasan Raheem's tour data and analyzes it using SQL, Python and Excel as part of my Spotify portfolio case study.

## Files
- `scraper.py` → Python script for scraping tour data
- `hasan_raheem_tour.csv` → Raw dataset of tour dates and venues
- `queries.sql` → SQL queries for practice and analysis
- `dashboard.xlsx` → Excel dashboard with charts and insights

## How to Use
1. Run `scraper.py` to generate the CSV file.
2. Import `hasan_raheem_tour.csv` into MySQL or Excel.
3. Use `queries.sql` to explore the dataset.
4. Open `dashboard.xlsx` to view visualizations.

## Purpose
This project demonstrates:
- Web scraping with Python (BeautifulSoup, Pandas).
- SQL queries for filtering, grouping, and joining.
- Excel dashboards for visualization.
- A case study benchmarking Hasan Raheem’s tour data for diaspora and mainstream audience expansion.
